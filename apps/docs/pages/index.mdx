## COMMANDS FOR ALL WORKSPACES

- run all workspaces in dev mode <Code>yarn dev</Code>
- start all workspaces in production mode <Code>yarn start</Code>
- build all workspaces <Code>yarn build</Code>
- lint all workspaces <Code>yarn lint</Code>
- test all workspaces <Code>yarn test</Code>

## COMMANDS FOR SINGLE WORKSPACES

- run a single workspace in dev mode <Code>yarn workspace -name- dev</Code>
- start a single workspace <Code>yarn workspace -name- start</Code>
